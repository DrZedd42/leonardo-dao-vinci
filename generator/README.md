
# DAO Vinci Generator

## Buidl

```bash
docker build . -t dao_vinci_generator
```

## Run

```bash
docker run dao_vinci_generator python /main.py
docker run --rm -v ~/output:/code/res dao_vinci_generator python ./main_line.py
```
